# RPG Game - エリア移動システム

日本語RPGゲームのプロジェクト。Pygame を使用して実装されています。

## 主要機能

- **エリア移動システム**: エリア間の視覚的に連携された移動
- **優先度制御**: エリア移動ポイント > エンカウント
- **戦闘システム**: ターンベースの戦闘
- **キャラクター管理**: プレイヤーと敵のステータス管理

## 実行方法

```bash
python main.py
```

## 操作方法

- ↑←→↓: 移動
- SPACE: アクション
- z: 神威発動
- x: メニュー
- s: セーブ (セーブポイント)

## 要件

- Python 3.8+
- Pygame

```bash
pip install -r requirements.txt
```
